 # index.ts
   Creates and exports a Vue I18n instance for internationalization with English, French, Finnish, and Swedish translations.

   ## Purpose
   This module facilitates the localization of Next Step's user interface to support multiple languages, enhancing accessibility and usability.

   ## API
   - **i18n**: Vue I18n instance with English as the default locale and fallback.

   ## Usage
   ```typescript
   import { i18n } from './index'
   // Use i18n in your components for translation
   ```

   ## Dependencies
   - **Locales**: en.json, fr.json, fi.json, sv.json
